# Kids Practice Hub

Upload these files to the root of the GitHub repository named `games`.

## File structure

- `index.html` is the main practice hub.
- `evan-vocabulary/index.html` is Evan's vocabulary game.
- `eli-sight-words/index.html` is Eli's sight word recognition game.
- `assets/styles.css` controls the look of all pages.
- `manifest.json` helps the site behave more like an app when added to an iPad home screen.

## GitHub Pages setup

1. Open the `games` repository in GitHub.
2. Go to Settings.
3. Click Pages.
4. Source: Deploy from branch.
5. Branch: main.
6. Folder: /root.
7. Save.

Your site should become available at:

`https://YOUR-GITHUB-USERNAME.github.io/games/`

Open that link in Safari on the iPad, tap Share, then Add to Home Screen.
